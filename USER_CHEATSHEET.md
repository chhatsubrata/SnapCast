# 📸 Shot Capture — 1-Page Cheat Sheet

> A floating bubble that **predicts when the next screenshot will happen** and shows a live countdown.
> It only *guesses* from past patterns — it never takes, blocks, or touches screenshots. Estimates, not promises.

---

## ▶ Get Started in 30 Seconds
1. **Open** the app → glassy bubble appears on top of your screen.
2. **Run the Wizard** → pick **Manual** → **Start Learning**.
3. Each time a screenshot happens, **click the record button** a few times.
4. Watch the **countdown + colors** predict the next one.

---

## 🪟 The Floating Bubble
| You see | Meaning |
|---|---|
| **08:42** big number | Time to next *estimated* screenshot |
| **Ring color** | 🟢 safe → 🟡 → 🟠 → 🔴 very soon |
| **Risk %** | Chance of screenshot *right now* |
| **Confidence %** | How sure the app is |
| **Badge** | ⚪ No Data · 🟡 Learning · 🟢 Tracking · 🔵 High Confidence |

- **Drag** = move it (remembers spot).
- **Double-click** = switch Expanded ↔ Compact.

---

## 🔌 3 Ways It Learns (pick in Wizard)
- **Manual** — *easiest.* You click a button each screenshot.
- **Folder Watcher** — point to your screenshots folder; auto-detects new images.
- **Log Watcher** — advanced; reads a log file.

---

## 🚨 Alert Levels
| Level | Look |
|---|---|
| **SAFE** | Calm |
| **WARNING** | Soft glow / pulse |
| **HIGH** | Orange border, strong pulse |
| **CRITICAL** | Red glow + notification — screenshot any second |

---

## ⚙ Useful Settings
- **Auto-Hide** — bubble vanishes at high risk, returns later (default: hide at **95%**, back in **15s**).
- **Theme** — Dark / Light / System · **Opacity** · **Widget size**.
- **Notifications** — on/off + sounds.
- **Launch on startup** — opens with your computer.
